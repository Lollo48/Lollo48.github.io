Game Overview

Sandwich is a mobile game developed in Unity 3D that challenges players' logic and precision through a grid-based positioning system and swipe controls. The goal is to assemble a perfect sandwich layer by layer, following specific ingredient sequences.

Game Goal

The player must correctly stack the sandwich ingredients in the right sequence. By using swipe gestures, they move the ingredients into position on the grid. Completing the recipe correctly allows the player to progress through the levels.

Key Features

  • Grid-based positioning: Ingredients must be moved and stacked according to the grid structure.
  • Swipe-based controls: Players use swipe gestures to move ingredients in specific directions.
  • Level design: Each level introduces new challenges, increasing complexity and requiring greater precision.

Gameplay

The player must drag ingredients along a grid using swipe gestures in four main directions. Ingredients must be placed correctly to form the sandwich in the predefined sequence. Incorrect placements require reorganization or restarting the level.

Game Mechanics

  • Grid management: Handles ingredient positioning and collision detection.
  • Swipe controls: Gesture-based movements to shift ingredients.
  • Level progression: Increasing difficulty as the player advances through levels.
  • Combination validation: Ensures ingredients are stacked in the correct order before finalizing the sandwich.
  • Grid Manager: Implemented the system for positioning ingredients on the grid, handling collisions, and checking placements.
  • Swipe Input: Developed the swipe control system to allow smooth and precise ingredient movement.
  • Level Manager: Designed level structure and managed progressive difficulty.
  • Game Logic: Created the logic that verifies the correctness of the ingredient sequence before validating the sandwich.
