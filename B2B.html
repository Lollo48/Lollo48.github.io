Game Overview

Museum is a mobile application developed using Unity 3D that offers an interactive and educational virtual tour of a museum. The app allows users to explore various exhibits and artworks in a museum setting from the comfort of their own device.

Game Goal

The primary goal of Museum is for users to reconstruct all the artworks within the virtual museum. By doing so, players unlock detailed information about each piece and earn rewards that include discounts for real-life museum visits. Additionally, the player can scan artworks on-site to unlock additional information via the Vuforia system.

Key Features

  • Virtual Museum Exploration: Navigate through a detailed 3D representation of a museum, exploring different rooms and exhibits.
  • Interactive Exhibits: Tap and interact with various artworks and artifacts to learn more about their history and significance.
  • User-Friendly Controls: Intuitive touch controls for smooth navigation and interaction, optimized for mobile devices.
  • Interactive Art Reconstruction: Players can interact with various paintings, noticing missing elements. They must search throughout the museum to find and collect these missing pieces. Once the artwork is fully restored, players unlock new information and insights about the piece.
  • Rewards System: By reconstructing all the artworks within the museum, players can unlock exclusive information about each piece and earn discounts that can be used for real-life visits to the museum.

Gameplay

Players navigate through the museum using simple on-screen arrows that guide them from one location to another. The intuitive navigation system ensures that players can easily move through the museum's different rooms and sections. Players can tap on various artworks and artifacts to view detailed descriptions and learn about their significance. Throughout the museum, players will find artworks with missing pieces. They need to search for these pieces and collect them. Once an artwork is fully restored, players unlock additional information and insights about the piece.

Game Mechanics

  • Exploration: Navigation, Movement, and Environment interaction.
  • Interaction: Tapping to access and reconstruct artworks, as well as retrieving information.
  • Saving Progress: The game supports saving the player's progress, ensuring no loss of data between sessions.
  • Login and Player Saving: User accounts and profile management allow players to log in and manage their game data.

Personal Contribution

  • Camera Manager/Controller: Designed and implemented the camera manager and controller to ensure smooth and dynamic camera movements within the virtual museum. This system allows the camera to follow the player's movements.
  • Input Manager: Created the input manager to handle touch controls and user interactions, ensuring a responsive and intuitive control scheme on mobile devices.
  • Inventory Management and UI: Implemented the inventory system, allowing players to collect and manage items such as missing pieces of artworks. The UI displays collected items and allows easy management.
  • State Pattern for Game States: Applied the state pattern to manage different game states (exploring, interacting, and reconstructing artworks), ensuring efficient organization of game logic.
  • Dialogue System: Created a system to manage in-game interactions and conversations, providing context and additional information about the artworks and artifacts.
  • Map Controller: Implemented a system allowing players to view and navigate the museum map, assisting in exploration.
  • Pause Menu: Developed the pause menu, enabling players to pause the game, access settings, and manage their progress.
  • Vuforia Management: Integrated Vuforia, an augmented reality (AR) platform, allowing players to unlock additional information by scanning artworks with their devices.
  • Animations: Created various animations, including animations for doors and other objects within the museum.
  • Sound Management: Managed the implementation of sound effects and background music, enriching the immersive experience.
